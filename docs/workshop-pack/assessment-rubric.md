# Assessment Rubric

| Area | 1 | 2 | 3 |
|---|---|---|---|
| Folder discipline | Wrong folder or unclear files | Correct folder but weak explanation | Correct folder and clear file roles |
| Public-safe data | Private data risk present | Synthetic data mostly used | Synthetic boundary clear and maintained |
| Eval practice | Did not run eval | Ran eval but cannot explain result | Ran eval and explained pass/fail cause |
| Safety behavior | Risky questions not separated | Some risky cases included | Fallback, handoff, and safety stop are explicit |
| Validation evidence | No log | Partial log | Commands, result, evidence, and remaining risk recorded |
| Handoff readiness | No next owner context | Basic summary | Clear owner, current state, limits, and next step |

Completion requires at least `2` in every area and `3` in public-safe data.
