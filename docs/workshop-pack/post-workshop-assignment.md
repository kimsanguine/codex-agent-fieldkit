# Post-Workshop Assignment

Complete one small adaptation with synthetic data.

## Task

1. Choose one internal workflow category.
2. Rewrite it as 5-7 fictional synthetic FAQ rows.
3. Add 5 golden-set cases:
   - 2 happy-path cases
   - 1 fallback case
   - 1 safety-stop case
   - 1 human-handoff case
4. Run the validation loop.
5. Update validation log and handoff notes.

## Submit

- Changed file list
- Eval score
- Validation report
- Handoff summary
- Remaining risks

## Do Not Submit

- real customer data
- internal URLs
- screenshots
- credentials
- private policy or process documents
