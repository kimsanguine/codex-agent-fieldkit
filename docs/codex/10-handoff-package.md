# 10. Handoff Package

A useful agent prototype is not done when the demo runs.

It is done when another owner can understand:

- what exists
- how to run it
- what was verified
- what is unsafe or incomplete
- who owns the next step

## Files

```text
_handoff/handoff.md
_handoff/migration_checklist.md
docs/validation_log.md
```

## Ask Codex

```text
Update the handoff package based on the latest validation result.
Be explicit about what is not production-ready.
```

## Handoff Must Not Include

- credentials
- private accounts
- browser sessions
- private workspace links
- raw chat transcripts with sensitive context
