# LinkedIn Launch Draft

비개발자가 Codex로 AI Agent를 만들 때, 핵심은 프롬프트가 아니었습니다.

진짜 차이를 만든 것은 운영체계였습니다.

- 올바른 폴더 구조
- `AGENTS.md` 프로젝트 메모리
- synthetic data
- golden set
- validation log
- public-first safety
- handoff package

그래서 공개 가능한 형태로 정리했습니다.

`Codex Agent Fieldkit`

비개발자가 Codex로 AI agent를 만들고, 검증하고, 넘길 수 있게 만든 unofficial fieldkit입니다.

첫 버전은 의도적으로 작습니다.

API key 없이 돌아가는 FAQ starter kit 하나, 그리고 `make demo`, `make test`, `make eval`, `make validate`, `make validate-report`로 검증되는 흐름 하나를 넣었습니다.

이번 업데이트에는 다음도 추가했습니다.

- README hero image
- Remotion intro video
- validation report artifact
- starter-kit template
- 보험/서비스 운영 합성 pack
- 워크숍 worksheet와 facilitator pack
- enterprise IT preflight와 signoff templates

프롬프트 모음이 아니라, 작동하는 작업면을 공유합니다.
