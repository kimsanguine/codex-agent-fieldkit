# Fieldkit Intro Video

Remotion source for the repository intro video.

## Commands

```bash
npm install
npm run still
npm run render
```

Rendered files:

- `assets/video/codex-agent-fieldkit-intro-frame.png`
- `assets/video/codex-agent-fieldkit-intro.mp4`

This project is isolated from the Python starter kit so the repo remains
offline-runnable for non-video workflows.
