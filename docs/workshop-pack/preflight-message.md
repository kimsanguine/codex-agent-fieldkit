# Preflight Message

Subject: Codex Agent Fieldkit workshop setup

Before the workshop, please prepare the following:

1. Confirm that you can access Codex.
2. Confirm that you can open a terminal.
3. Confirm that Python 3 and Git are installed.
4. Clone or download the workshop repo through the approved channel.
5. Do not bring customer records, internal links, screenshots, credentials, or
   private policy documents to the exercise.

We will use a fictional company and synthetic data only. The goal is to practice
the delivery loop: folder, memory, data, eval, safety scan, validation report,
and handoff.

If setup fails, bring the error message to the workshop channel. Do not paste
credentials or private data.
